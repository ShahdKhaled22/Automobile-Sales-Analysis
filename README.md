# 🚗 Automobile Sales Analysis During Recession Periods

## 📋 Project Overview
Analysis of historical automobile sales data for XYZAutomotives to understand 
how sales were affected during recession periods.

## 📊 Part 1: Visualizations
Built using Matplotlib, Seaborn & Pandas:

| Task | Description |
|------|-------------|
| 1.1 | Line chart — Automobile sales fluctuation year to year |
| 1.2 | Advertising expenditure vs sales (non-recession) |
| 1.3 | Sales trend per vehicle type (recession vs non-recession) |
| 1.4 | GDP variations during recession vs non-recession |
| 1.5 | Bubble plot — Impact of seasonality on sales |
| 1.6 | Scatter plot — Vehicle price vs sales volume |
| 1.7 | Pie chart — Advertising expenditure share |
| 1.8 | Pie chart — Ad expenditure per vehicle type |
| 1.9 | Unemployment rate effect on sales |

## 📈 Part 2: Interactive Dashboard
Built using Plotly & Dash:
- Recession Period Statistics Report
- Yearly Statistics Report

## 🛠️ Technologies Used
- Python, Pandas, Matplotlib, Seaborn
- Plotly, Dash
